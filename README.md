## My Portfolio

This repository contains the source code for my personal portfolio website. It showcases my recent experience, technical skills, and projects, including applications I have developed using API integrations, real-time features, and modern frontend development practices.

The portfolio serves as a space to document my work, track my growth as a developer, and share my projects with others. It reflects my learning journey, the technologies I enjoy working with, and the solutions I have built while exploring full-stack development.

## Tech Stack

**Frontend** : React, Vite, CSS  
**Backend** : Node.js, Express, MongoDB  
**Email** : Nodemailer via Gmail SMTP  
**Database** : MongoDB Atlas  

## Setup

```bash
# Frontend
npm install
npm run dev

# Backend
cd backend
npm install
node server.js
```
